garage2.java (an improvement of garage.java)

description:
Class car for creating car instances (objects).
Class garage2 is the main program (has the main method).
Method car.display() formats the output with columns (like a table).

compile:
c:\java_book\Labs\002> javac garage2.java

run:
c:\java_book\Labs\002> java garage2
or
c:\java_book\Labs\002> java -Dfile.encoding=737 garage2
(to see Greek characters)
